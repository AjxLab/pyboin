pyboin
======

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)


pyboin is Pure-Python Japanese character interconverter for Hiragana and Vowel.
```
「こんにちは」 -> 「おんいいあ」
```


## Requirement
* Python3


## Installation
```sh
$ pip install pyboin
```
